## 테이블 생성 시 primary key 지정 2가지 방법

```sql
// 기본
CREATE TABLE bookinfo_tbl (
    bookno NUMBER(6) PRIMARY KEY, // 자동으로 제약조건이 primary key + not null 포함
    author CHAR(24),
    cookname CHAR(24)
);

// 제약조건 추가
CREATE TABLE bookinfo_tbl (
    bookno NUMBER(6),
    author CHAR(24),
    cookname CHAR(24),
    CONSTRAINT pk_bookinfo PRIMARY KEY (bookno)
);
```

## 테이블 & 컬럼

**테이블 컬럼 네임 확인** - DESC reservation_tbl;  
**컬럼 네임 변경** - ALTER TABLE bookinfo_tbl rename column cookname to bookname;  
**컬럼 타입 변경** -
ALTER TABLE reservation_tbl MODIFY (bookno VARCHAR2(10));  
**컬럼 추가** -
ALTER TABLE reservation_tbl ADD (remark VARCHAR2(100));  
**컬럼 삭제** -
ALTER TABLE reservation_tbl DROP COLUMN remark;  
**컬럼 기본값 변경** -
ALTER TABLE reservation_tbl MODIFY (status DEFAULT '1');
